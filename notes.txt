<absolutePATHtoDESTINATIONdirectory> with the complete path to the destination directory on the cluster. You can find this path using the pwd command (which prints the working directory) on the cluster after navigating to the desired destination directory.

<absolutePATHtoFILE> with the complete path to the file on the cluster. You can find this path using the realpath file_to_be_copied command on the cluster.

You can submit a job writing a script (script_name.sh) and submitting it with the command:

$ sbatch script_name.sh
#!/bin/bash
#SBATCH --job-name="test_submission"
#SBATCH --account=<YourStudentIdNumber>
#SBATCH --partition=stud
#SBATCH --qos=stud
#SBATCH --output=out/%x_%j.out # %x gives job name and %j gives job id
#SBATCH --error=err/%x_%j.err
echo "Who am I?"
whoami
echo "Where ?"
hostname
sleep 10

SLURM directives

#SBATCH --partition=<partition_name>
#SBATCH --partition=stud

SLURM Notification by Email

#SBATCH --mail-type=END
#SBATCH --mail-user=3202029@studbocconi.it
(Other valid options are: BEGIN, FAIL, ALL).

After submitting the job using the SLURM command sbatch you can check the status of your job in the queue by typing:
$ squeue -u <username>

To get more information about a job:
$ scontrol show job <jobID>

The command scancel allows to cancel a job with the jobID:

$ scancel <jobID>

You can cancel all the jobs for the user <my_username> with the command:

$ scancel -u 3202029
