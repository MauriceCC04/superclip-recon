SuperCLIP-Recon

Can image features recover missing caption tokens?
A COCO-scale test of whether lightweight masked-token reconstruction helps a SuperCLIP-style image-text retrieval baseline.
Research question. Can masked caption-token reconstruction provide useful additional supervision beyond token classification in a COCO-scale SuperCLIP-style retrieval model?

Where we start

From CLIP to a testable gap.

  • CLIP. Learns global image–text alignment by making matching image-caption pairs close in embedding space.
  • SuperCLIP. Keeps CLIP's contrastive loss and adds token classification: from the image alone, predict which caption tokens are present.
  • Gap. Token presence is bag-of-words-like. It can capture which concepts appear without fully capturing how they relate.
  • Our test. Add masked-token reconstruction from image features and measure whether it improves retrieval or compositional diagnostics.

Dataset & setup

Why MS-COCO Captions.

COCO is a natural course-scale benchmark: paired images and multiple human captions per image, enabling the standard 5-caption-per-image image-text retrieval evaluation.

Why this is lightweight
  • One MLP head, no decoder, no extra backbone, no extra dataset.
  • Image features alone predict the masked tokens — no text context.
  • Observed runtime overhead ≈ 0.9% (3648 → 3680 s).
Main result

Cheap to add; retrieval preserved; improvement uncertain.

What the data supports

Strongly supported

  • ReconA is feasible, lightweight, retrieval-compatible.
  • Runtime overhead ≈ 0.9%, no decoder, no extra data.

Weakly supported

  • A small mean retrieval gain in this 5-seed COCO setup (Δ R-sum = +0.317).
  • Small positive shifts on several probes, but mixed overall.

Not supported

  • A statistically established retrieval gain — 95% CI crosses zero.
  • A robust improvement in compositional reasoning — ARO Relation regresses.
  • Original-scale SuperCLIP behavior — this is COCO-only with a top-1000 token-classification vocabulary.

Exploratory only

  • Variant B — phrase reconstruction, 1 seed, retrieval only.
  • λ and mask-rate sweeps — ≤ 2 seeds each.
Next steps: larger-scale runs · full-vocabulary SuperCLIP-style token-classification targets · structured phrase masking with paired seeds · stronger compositional benchmarks (e.g. SugarCrepe).
